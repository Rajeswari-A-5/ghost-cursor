## 👻 What is Ghost Cursor?

> A **delightful browser toy** where a hand-crafted SVG ghost smoothly follows your cursor with a **5-dot trailing animation** — and you can change its color live with a built-in **color picker**. No libraries. No frameworks. Pure vanilla magic.

<br/>

## ✨ Features

<div align="center">

| | Feature | Description |
|---|---|---|
| 👻 | **Animated Ghost** | Hand-crafted SVG with eyes, blush & a cute smile |
| 🖱️ | **Cursor Tracking** | Smooth lag-based real-time mouse following |
| 🎨 | **Live Color Picker** | Change the ghost's body color instantly |
| 🫧 | **Trail Effect** | 5 glowing dots fade behind the ghost as it moves |
| 💅 | **Zero Dependencies** | Pure HTML + CSS + JavaScript — nothing to install |

</div>

<br/>

## 🎬 How It Works

```
  🖱️  Move mouse
        │
        ▼
  📍  Capture X, Y position (mousemove event)
        │
        ├──▶  👻  Ghost position updates with smooth lag
        │
        ├──▶  🫧  5 trail dots follow with staggered delay offsets
        │
        └──▶  🎨  Color picker updates SVG fill in real-time
```

## 🌈 Customization Ideas

Want to extend it? Here are some fun ideas:

- 🔮 **Custom ghost shape** — edit the SVG `<path>` in `index.html`
- 👁️ **Blinking eyes** — add `@keyframes blink` on the eye `<circle>` elements
- 🔊 **Spooky sounds** — trigger audio on `mousemove` or `click`
- 🌙 **Dark / light toggle** — swap the page background theme
- 💨 **More trail dots** — add extra `.trail` divs and handle them in `script.js`
- 🌀 **Ghost spin on click** — add a `click` listener that applies a CSS rotation
